      HaircuttersCalculator doing all calculation logic.

      Depends on paramaters like population HaircuttersCalculator will approximate number of haircutters in this country. There are methods to change default parameters and get results.

      calculate() : integer
      This is not exactly number, rather an demand for hairdressers in specified circumstances. Formula: mans = population * ( man_rate / 100 ); womans = population - mans; time_needed = (mans*shear_time_man + womans*shear_time_woman); time_have = haircutter_have_days * haircutter_have_minutes; result = ceil( time_needed * shear_per_year / time_have ) ;
